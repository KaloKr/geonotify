# GeoNotify

## Описание

GeoNotify е уеб приложение, което използва съвременни уеб технологии за работа с локация и известия.

Приложението позволява на потребителя да:

* получи текущата си географска локация
* получи известие при успешно засичане на локация
* види координати (ширина и дължина)
* получи приблизителен адрес
* отвори локацията си в Google Maps
* намери ресторанти в близост до себе си

---

## Използвани технологии

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Geolocation API
* Notification API
* OpenStreetMap (Nominatim API) – за преобразуване на координати в адрес
* Google Maps (URL integration)

---

## Как работи

1. Потребителят натиска бутона „Намери ме“
2. Браузърът иска разрешение за достъп до локацията
3. При успешно засичане:

   * показва координати
   * показва точност
   * извлича адрес чрез външно API
   * изпраща известие
  
4. Потребителят може:

   * да отвори собствената си локация на карта
   * да търси ресторанти наблизо
